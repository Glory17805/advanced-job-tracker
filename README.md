# Advanced Job Tracker

A FastAPI-based backend project for managing job seekers, recruiters, jobs, applications, and resumes.

## Completed Modules
- Project setup
- SQLite database setup
- User Registration API
- User Login API

## Tech Stack
- FastAPI
- SQLite
- SQLAlchemy
- Pydantic
- JWT Authentication

## Run Project
uvicorn app.main:app --reload
